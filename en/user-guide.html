🛡️ WAC Shield PRO v3.0.9

Developed by WAC Security Solutions

1. Quick Installation on WordPress

  1. Go to Plugins > Add New in your WordPress dashboard.
  2. Upload the wac-shield-pro.zip file.
  3. Activate the plugin and find the WAC Shield menu in the sidebar.

2. Defense Settings

  • Geo-Block: Choose the authorized country (e.g., USA).
  • Brute Force: Set the maximum login attempts (Recommended: 3).
  • Base Ban Duration: Set the initial lockout interval (Recommended: 15 minutes). The duration increases automatically with each repeated attempt.
⚠️ ATTENTION: Always add your current IP address or authorized users to the Whitelist to prevent accidental lockouts.

3. Smart Updates

Whenever WAC Security Solutions releases a new security layer, you will be notified. To ensure you are running the most secure version, follow these steps:

  • ✅ Automatic Alert: A notification will appear in your Plugins list. Simply click "Update Now".
  • 🔍 Manual Check: If a new version was released but hasn't appeared yet, go to Dashboard > Updates.
  • 🔄 Force Search: Click the "Check Again" button. This clears the WordPress cache and forces a connection to our security cloud.
  • 📦 Plugin List: Scroll down to the plugin list, select WAC Shield PRO, and click "Update Plugins".
💡 Pro Tip: Keeping your plugin updated ensures that your blocked IP lists and defense algorithms are always synchronized with the latest threats.
